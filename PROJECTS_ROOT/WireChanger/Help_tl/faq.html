WireChanger FAQ

  • Q: I set up weight for some of my themes, but now I want to remove weight. How can I do that?
    A: Just set theme`s weight to 0 (zero). This will make such theme equal to all other 0`s themes

  • Q: I select clock '...', but nothing happes - wallpapers are changed correctly, but no clocks occured. What`s the matter?
    A: May be you don`t have flash player installed. Flash player is very useful thing, so it`s a good idea to get one for your system. Latest flash player can be found here (Macromedia Flash plug-in)

  • Q: WireChanger can`t make my desktop icons transparent. Why?
    A: If you are using active clocks, the problem is in the windows system itself. WindowsXP of certain editions are unable to depict transparent icons when Active Desktop is enabled. Microsoft confirmed this to be a bug. If you are not use active clocks - email us about this problem

  • Q: Sometimes WireChanger does not change anything on my desktop. OS - Windows XP
    A: Sometimes, when you hide or show desktop icons on WinXP throught system menus, XP can "forget" about wallpapaper. This can be easily fixed: go to the display properties and forcedly set WireChanger wallpaper once. After that image cycling must be fine

  • Q: I choose template for my wallpapers but nothing cool happens. Subsequent wallpapers looks like just a simple images
    A: In order to use some of our templates for your images, you must have Internet Explorer v 5.0 or higher to be installed on your computer + JavaScript must be enabled

  • Q: When I try to enable some template my desktop becoming totally white. Why?
    A: Check your Internet Explorer settings, may be you disable "Show images" checkbox in additional settings. Images must be enabled

  • Q: How to find weather location code for my city?
    A: This can be done through visiting http://www.gismeteo.ru. Find your city and click corresponding link. Then look at the address bar, you`ll see something like http://...../XXX.html; This XXX number is your city code.

  • Q: What does it mean - exceptional conditions?
    A: Exceptional conditions are situations in which wallpaper changer can interfere with work of other applications. Processing of new wallpaper is very CPU and memory consuming operation. During processing WireChanger can steal CPU resources from such important task as burning CD-R disk, for example. To avoid similar problems, WireChanger can temporarily disable image rotation, until CPU and memory usage falls back to the usual level.

  • Q: I want to see your program in my language. Is it possible?
