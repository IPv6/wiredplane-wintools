Характерные черты WireChanger

"Менеджер" рабочего стола
- Технология "Active desktop" не используется (есть исключения - часы).
- Картинки можно добавлять путем перетаскивания из других приложений.
- Поддержка форматов изображений: jpg, gif, png, bmp, wmf, ico.
- Поддерживает гигантские коллекции картинок
- Разные уровни качества "растягивания".
- Возможность добавления в список картинок HTML-страниц и адресов Интернета (URL).
- Папку с картинками можно добавить в список в один клик.
- WireChanger может сменить картинку при загрузке системы и при выходе из нее (чтобы не менять картинку, нажмите и удерживайте клавиши Shift, Ctrl или Alt при загрузке системы).
- Все картинки можно разгруппировать по темам, для каждой из которой можно задать значимость. Значимость определяет, как часто картинки из данной темы будут использоваться для фона рабочего стола по сравнению с картинками из других тем.
- На картинку можно добавлять всевозможные шаблоны.
- Можно легко создавать новые шаблоны (по сути - это простой HTML с дополнительными макросами).
- Каждый шаблон имеет настройки.
- Прогноз погоды для шаблона "календарь" (выключен по-умолчанию, включить можно в опциях календаря). Спасибо http://www.gismeteo.ru/. Для просмотра прогноза погоды хост www.gismeteo.ru должен быть доступен.
- Картинки можно посмотреть в главном окне и в диалоге "Открыть".
- Цвет фона рабочего стола задается в настройках.
- Рисунки для темы можно синхронизировать из специального каталога.
- Смена фона рабочего стола не будет произведена, если сейчас свободно мало оперативной памяти.
- Смена фона рабочего стола не будет произведена, если сейчас сильно загружен процессор (только в системах NT, 2000, XP!).
- На каждое действие можно назначить горячую клавишу (клавиша <WINDOWS> поддерживается).
- Возможно добавлять заметки на рабочий стол. Для добавления/редактирования нужно сделать двойной клик.
- На рабочий стол можно добавить идущие часы.
- Возможно загружать новые рисунки и часы.


Системные требования.
Для использования WireChanger без шаблонов и часов, дополнительных требований к системе нет - просто установите Windows :). Для шаблонов необходимо иметь установленным Internet Explorer 5.0 или выше, в котором включено отображение рисунков и JavaScript (эти опции включены по-умолчанию). Для часов необходимо иметь установленным Macromedia Flash player (сливайте плагин тут).
