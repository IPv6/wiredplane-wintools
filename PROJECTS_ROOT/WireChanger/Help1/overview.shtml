Программа WireChanger

"Менеджер" рабочего стола с часами и заметками
Изменяет фон рабочего стола, добавляя на него потрясающие эффекты. В комплекте с программой идут шаблоны (календарь, миксер картинок, листок-мемо и другие), которые легко можно настроить "под себя". WireChanger может оживить Ваш рабочий стол с помощью идущих часов (из огромной коллекции ColClocks) и прилипающих к определённому месту заметок. Встроенная поддержка всех специальных форматов изображений: gif, jpg, bmp и др. Дополнительная возможность - показ страниц HTML или адресов Интернета. WireChanger может автоматически применять спецэффекты к фону рабочего стола (Ч/Б цвет, затемнение, календарь) "на лету". С WireChanger Вы можете легко добавлять на рабочий стол прекрасные текстовые заметки (поддерживаются скины) - для добавления достаточно дважды кликнуть по свободному месту рабочего стола и ввести текст. Список используемых картинок можно разгруппировать по темам, для каждой из которых можно назначить значимость. Значимость определяет, как часто картинки этой темы будут использоваться по сравнению с картинками из других тем. Можно назначить каталог для синхронизации картинок и все новые рисунки будут добавлены в тему автоматически при запуске WireChanger.
На фон рабочего стола могут быть добавлены шаблоны. Шаблоны могут добавлять на изображение всевозможную полезную информацию (Например: прогноз погоды или цитаты знаменитых людей). Другой шаблон выводит на изображение календарь на 2 недели или целый месяц с возможностью добавления любого текста прямо в его ячейки! WireChanger может добавить на рабочий стол идущие часы из коллекции ColClock или http://www.adamdorman.com. Также можно загрузить небольшую программу "Desktop FlashGen" и создать интерактивный фон из любого flash-файла. Вы больше никогда не увидите одной и тойже картинки из Вашей коллекции дважды! :)


Интересные особенности
- Поддержка форматов изображений jpg, gif, png, bmp, wmf, ico.
- HTML-страницы, рисунки Интернета, веб-камеры, URL-адреса могут быть также добавлены и показаны.
- Папка с картинками и с подпапками может быть добавлена в список в один клик.
- WireChanger может проверять наличие новых рисунков автоматически.
- Весь вписок картинок можно разгруппировать по темам, темы контролируются независимо. Можно задать, как часто картинки из данной темы будут использоваться для фона рабочего стола по сравнению с картинками из других тем.
- Можно задать каталог для синхронизации картинок для каждой темы.
- На рабочий стол можно добавлять идущие часы из огромной коллекции.
- Фон рабочего стола радикально меняется, благодаря шаблонам, в результате чего он становится настоящей информационной панелью!
- Каждый шаблон имеет настройки, так что всю ненужную Вам информацию можно легко убрать с рабочего стола.
- Возможно добавление заметок поверх изображения. Заметки останутся на рабочем столе даже в том случае, если фон будет изменен из другого приложения, такого как Opera, ACDSee или IrfanView!
- Заметки добавляются очень просто - двойным кликом по свободному месту рабочего стола. Превратите WireChanger в персональный органайзер!
- Уникальный способ украшения рисунка - полу-отражение. Украшение будет отображаться каждые две ступени.
- Уникальные способы "пригонки" рисунка к размерам экрана - края рисунка будут "подогнаны" к краям рабочего стола с/без смесью с цветом фона.

Технические особенности
- Возможность предосмотра картинок и интуитивно понятный интерфейс.
- Фон рабочего стола можно менять вручную, по истечении определенного периода или при выполнении определенных условий (например "ScreenSaver активен").
- Изображения могут быть добавлены в список путем перетаскивания из других приложений или, собственно, из самой программы WireChanger путем указания определенной папки.
- WireChanger выявляет недостаточность оперативной памяти и высокую загруженность процессора. В таких случаях смена картинки будет отложена.
- WireChanger можно легко использовать, как персональный органайзер.
- Осуществляется полный контроль над активными элементами рабочего стола и иконками (цвет, прозрачность и т.д.).
- WireChanger может автоматически изменить фон рабочего стола, когда Ваша рабочая станция стала заблокированной.
- На каждое действие можно назначить горячие клавиши (клавиша <WINDOWS> поддерживается).
- Все окна "прилипают" друг к другу, когда находятся рядом.
- Разные уровни качества "растягивания".
- Каждая деталь может быть настроена отдельно.

Внимание: Обычная лицензия позволяет устанавливать WireChanger сразу на 3 компьютера.
Если Вы желаете зарегистрировать WireChanger бесплатно, создайте свой скин для заметки или переведите WireChanger на свой язык, а результат отошлите нам. В ответе на письмо мы пришлем регистрационный ключ в качестве благодарности за проделанную работу :)

Системные требования
Для использования WireChanger без шаблонов и часов, дополнительных требований к системе нет - просто установите Windows :). Для шаблонов необходимо иметь установленным Internet Explorer 5.0 или выше, в котором включено отображение рисунков и JavaScript (эти опции включены по-умолчанию). Для часов необходимо иметь установленным Macromedia Flash player (сливайте плагин тут).

P.S. Если у Вас есть хорошие "обои" для рабочего стола или созданный шаблон/скин заметки, поделитесь этим с другими пользователями - архив .zip можно выслать на wallpapers@wiredplane.com. Мы обязательно выложим ваш архив на вышеуказанном домене :^) Периодически ходите на http://www.wiredplane.com/wallpapers.shtml за обновлениями!
