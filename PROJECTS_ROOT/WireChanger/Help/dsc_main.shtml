Main Window


1Checkbox. If checked, theme will be used in image rotation
2Theme name. Double-click to edit
3Image path
4Theme stretching option:
'Do not resize' - images will not be resized
'Exact fit' - images will be stretched to fit the screen
'Proportional fit' - images will be stretched to fit the screen preserving proportions
'Automatic fit' - images will be stretched to fit the screen preserving proportions if they are big enough only (no stretching in other case)
5Theme weight (percent). Weight defines how frequently images from this theme will be used in image rotation.
Value 100 means that image rotation will use only this theme
Value between 0 and 100 means that image rotation will use this theme with corresponding frequency
Value 0 means that this theme will be used rarely (depending on other themes weights)
6Hours
7Minutes
8Save and close
9Checkbox. If checked, wallpaper will be changed after specified time (see 6 and 7)
10How to get next image from theme. 'In order' means that images will go one after another. 'Random order' means that images will go in random order
11Quick access to some common functions (from left to right) - add images (new only/full directories/etc), add folders with images, add image archive, delete theme, sort theme images, remove old and broken links to images
13Available templates list
15Preview window. Note checkbox with selected image file name.
If checked, every selected image will be previewed first. This may take some time for URL`s
16Selected image resolution information
17Close window (the same as 'OK' button)
