Tips and Tricks

Note/Message 
-You can attach note to underlaying window by clicking on attach button with Ctrl pressed
-Note in Icon mode takes much less place on the screen. You can quickly switch note to this mode by clicking on attach button or double-clicking on note title
-If you want to temporarily hide note and to see what is under note`s window, you can move your mouse over it with Alt pressed (corresponding option must be ON)
-You can attach file to the message by simply draging it from other application (note that files can be sended only through Xpc protocol)
-If you press Ctrl-Shift-Enter, you message will be sended to recipient and deleted if the case of success
ToDo 
-You can associate ToDo with application by defining activation rules upon that application. ToDo will be active all the time you work with such associated application
Reminder 
-You can right-click on watch window to set exact minute
-To create shared reminder you must check corresponfing item in action drop-down list at reminder`s window, shared reminders must be enabled in preferences
-To create reminder to trigger everyday except holidays, use "Skip reminder on specified days of week/month" option. Using this option you can set up days, on wich reminder should be silently skipped if triggered. So you can create reminder to display text every day, 9:00 and set it to skip itself at Sunday/Saturday
Main window 
-You can add link to any file by draging it from other application
-To move item to another folder or position, you can simply drag and drop it
-To quickly set item priority or color style: just right-click on first column
-You can sort items by clicking on column titles
-Double click on item`s creation date or priority to quickly modify/activate item
-To rename item`s title just double-click on it. For non-empty folders and links you can select corresponding menu item from context menu
Address book 
-To move person to another position, you can simply drag and drop it
-To view message history for one person only, select 'Message history' from person`s context menu
-To open person computer in explorer, just double-click on 'Computer name' column
Command line switches 
-To import settings from another computer, run WN with -io command line switch
-To import address book from another computer, run WN with -ia command line switch
