WireKeys Overview

WireKeys is an all-in-one tool for power users for automating common operations with text, applications, your desktop, and many Windows features. Define hotkeys using any keyboard combinations you can devise. You can even include mouse button clicks and wheel spins (if your mouse is so equipped) or remote buttons. You can automate and virtually eliminate almost all of your daily, repetitive tasks - file and directory manipulation, windows manipulation, text replacement, text processing, file uploads and downloads, and more. Extend WireKeys' functionality with additional plugins. The four most useful plugins (Window's traits, Autotext, special hotkeys support and Macro recorder) are already included in the installation package.
You can learn more by looking at Features
