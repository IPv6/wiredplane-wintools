ver 3.1.1
Added:
- New hotkey type (behaviour) - "Sequence". Hotkey of this type will be triggered when you type keys exactly in the same order as they were added to hotkey. Example: create 'sequence' hotkey "Ctrl-W-O-R-D". Then press key 'Control' (and unpress it), 'W' (and unpress it),'O' (and unpress it),'R' (and unpress it),'D' (and unpress it), in this order to activate it. Something like typing hotkey in invisible console (regardless of active application)
- 'Pause keyboard' hotkey in the 'Hotkeys' section of preferences. This action will block all keyboard buttons, but leave hotkeys intact and active. You will be able to press hotkeys even when keyboard is paused. Additionaly, WK can automatically remove pause when hotkey is triggered. Useful in conjuction with "Sequence" hotkeys. Pause key will be automatically added to all new sequence hotkeys by default for your convinience. See help for details
- New hotkey option - 'Press+Unpress'. Hotkey with this option will be triggered twice - when pressed and when unpressed. Can be used in conjuction with "Sequence" hotkeys.

Action will be triggered as usual only when you unpress hotkey combination. Example: Suppouse Win-K is assigned to "Show preferences" and Win-K-L is assigned to "Pause WinAmp". If Win-K is not marked as "Act on unpress", pressing Win-K-L will show preferences and pause WinAmp at the same time (because while you are pressing Win-K-L, you are pressing Win-K too). But if you mark Win-K as "Act on unpress", Win-K will show preferences only when pressed and unpressed alone, and will not trigger event after unpressing Win-K-L (because Win-K-L triggering another action).
- Hotkey now has another new property - Transparency. Transparent hotkeys will not hide keypress from underlaying application. So it is possible to reload such combination like Ctrl-C for example - and currently active application will receive this keypress too and will copy it`s selected content into clipboard

2) WireKeys.exe -ul=XXX
WireKeys will save its preferences in the folder "XXX" inside folder with WireKeys.exe file
3) WireKeys.exe -ul="C:\XXX"
WireKeys will save its preferences in the folder "C:\XXX"
