Быстрый запуск приложений

Используя WireKeys Вы можете запускать любое приложение просто нажав комбинацию клавиш или нарисовав жест мышью. Для задания настроек и комбинация служит окно, открываемое пунктом "Приложения/макросы" трей-меню
  • После запуска можно назначить приложению свернуться в трей, развернуться на весь экран или минимизироваться.
  • Окну приложения может быть автоматически выставлен размер, положение и состояние "Поверх всех".
  • Приложение может быть запущено как на активном, так и на любом другом дополнительном десктопе.
  • Для удобства каждому приложению быстрого запуска может быть выделена отдельная иконка в трее или отдельное плавающее окошко.
  • Также при старте приложения есть возможность активации уже запущенного экземпляра запускаемой программы вместо запуска еще одной копии.
  • Если вы добавите в быстрый запуск каталог диска, то у Вас будет возможность просматривать этот каталог (т.е. его содержимое - файлы, подкаталоги и т.п.) из меню в трее или меню плавающего окошка - по правой кнопки на нем.
  • На плавающее окошко можно переносить файлы - они автоматически будут копироваться в соответствующий иконке каталог. Если плавающее окошко представляет из себя не папку, а приложение (или макрос), то оно будет запущено с параметром, состоящим из путей к перетащенным файлам (другими словами перетащенные файлы будут открыты этим приложением).

    Каждый пункт быстрого запуска может иметь категорию. Категории также может быть назначена горячая клавиша, вызов категории покажет меню с элементами, в нее входящими. Двойное нажатие гор. клавиши категории будет означать вызов всех пунктов быстрого запуска, в ней находящихся
    Пункты быстрого запуска всегда можно вызвать не пользуясь горячими клавишами - через меню в трее или дополнительное меню - или назначить различным пунктам одну и туже горячую клавишу (тогда при нажатии они все будут запущены одновременно) Вы также можете активировать набрав заданный текст ("синоним") в любом приложении. Для задания этого текста-синонима раскройте пункт с соответствующей горячей клавишей в настройках Обратите внимание: эти пункты меню имеют свое собственное контекстное меню, которое вызывается нажатием на пункте меню правой кнопки мыши. Из этого дополнительного меню можно удалить пункт, скрыть его или выделить в отдельную иконку или открыть настройки именно этого пункта быстрого запуска
    Для создания нового пункта быстрого запуска можно воспользоваться одним из нижеприведенных способов:
    - Открыть настройки и в соответсвующей закладке нажать на кнопку создания пункта быстрого запуска нужного типа. Это единственный способ создавать пункты типа "Каталог"
    - Нажать правой кнопкой мыши на заголовок окна, которое вы хотите добавить в быстрый запуск и выбрать из появившегося меню "Add to quick-run"
    - Вызвать дополнительное меню активного окна (либо нажав правой кнопкой мыши на крестик в верхнем правом углу окна либо нажав горячую клавишу - 'Ctr-Alt-Grey Minus' по умолчанию) и в подменю "Ярлык приолжения" выбрать "Добавить в быстрый запуск". В этом же меню можно создать ярлык к активному окну на десктопе или меню кнопки "Start"
    Специальное использование пунктов быстрого запуска:
    Вы можете вызывать пункты быстрого запуска/макросы из среды Windows. Используйте параметр строки "-execute" для запуска макросов или действий из bat/cmd/lnk файлов. Командная строка вида "WireKeys.exe -execute=XXX" где XXX - имя макроса/пункта быстрого запуска - запустит его на выполнение. Для выполнения действия задайте его номер предваренный символом '!', например "WireKeys.exe -execute=!15" выдвинет диск CD-Rom. Номера действий смотрите ниже. Внимание: WireKeys должен быть уже запущен чтобы выполнять такие действия.

    Вы также можете указать название пункта быстрого запуска/макроса/действия (номер после символа "!") в плагине "Автотекст", при назначении короткого имени запуску программы. Тогда при наборе короткого имени будет запущен макрос/пункт быстрого запуска/действие имя которого Вы указали
