WireKeys Plugins

Plugins are small additional applications, which can add functionality to the main application. Any plugin can be "unplugged" by user, so it is YOU who decide what must take your system resources and what is not. WireKeys keeps track and remember user-defined hot keys for every action of every plugin. Hotkeys and plugin`s preferences can be changed in the "Plugin options" window. Some plugins are disabled by default, you have to enable them in the preferences in order to use them
Short description of the default plugins:

- Wired key combinations. Do you use right-shift key for capitalizing your text input? And what about Caps Lock key? 80% of users have the same answer - Almost never! Now you can use such keys separately in the hotkey combinations as well as Caps Lock key and all multimedia keys. This plugin incredibly increases "hotkeyness" of your keyboard. With this plugin you can set different hotkeys for left- and right-shift/control/alt/win keys, use restricted key combinations for hotkey (such as Alt-Tab), use all additional keys of the multimedia keyboards and use multi-key hotkeys, like LCtrl-A-B-C or Space-Enter for example. You can also use your mouse buttons (including MS XButtons) and mouse wheel in your hotkey combinations as well. To add mouse button to hotkey, mouse pointer must be over hotkey input field

Options of this plugin can be found under "Global options" of the preferences dialog, in the "Hotkeys" section

- WireKeys windows traits. This plugin extends your abillity to control windows in the system. For example it makes all windows in your system sticky and allows you to hide windows to tray by right-clicking its minimize button. 'Sticky' mean that window will be automatically snapped to other windows edges and screen boundaries when moved. Stickness can be temporaly disabled by holding Shift key while moving window. Other plugin features:
- ability to define windows to be settled in tray instead of taskbar (such windows will be automatically moved to tray on focus blur)
- ability to force all windows to be inside screen boundaries (so you will never cry on window with its title bar placed beyond monitor edge ;)
- ability to call 'Hot menu' by right-clicking 'Close' button ('X' button) or from the window`s system menu (Alt-Space menu). A lot of useful stuff can be found here - application shortcuts/selected text processings/macros and more.
- ability to inject usefult items (add to quick-run, hide to tray, etc) into window`s system menu (alt-space or right-clicking window`s title)
- ability to dock/expand window in horizontal/vertical direction on right/middle-clicking window`s border (try to right-click or middle click windwo`s border to get the idea)

- Macro recorder. Using this plugin you can easily record keystrokes to replay them later by hotkey. You can also record macro that will replay different keystrokes for different applications. Simple example: you want to record "Open new media file" macro that will work in different media players - WinAmp and Media Player - in the same manner. This can be done by sending different keystrokes - 'L' for WinAmp, 'Ctrl-O' for Media Player - to player that is currently active. To record such macro choose corresponding option in the pre-recording dialog, switch to WinAmp and press 'L' then switch to Media Player and press 'Ctrol-O'. After that click tray icon to finish recording - and you will get what you want, new macro that will send 'L' key when WinAmp is active and 'Ctrl-O' when Media Player is in foreground


- WinLIRC remote control. Add any button from your remote to your hotkeys. WinLIRC must be up and running. Native support for AverTV cards


- Desktop icons font master. This plugin allows to set font and color of the desktop icons and was designed as free and simple sample for plugin developers. Here you can find full source code for it (compiled and tested on MSVC++ 6.0 Sp5)

Developer info:
Plugin is the simple DLL file (renamed to '*.wkp') or stand-alone '*.exe', which must reside in the "Plugins" folder and export certain functions. You can write your own plugin and send it to us to share with all WireKeys users in exchange for free registration key. Of course, you will be mentioned as author everywhere.
You can look at full source codes (MSVC++ 6.0 Sp5) for several plugins as an example.
You can also look at the plugin`s header file for MS Visual Studio C++ 6.0.
If you have any questions, you can post them in our support forum.
