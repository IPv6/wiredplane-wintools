Часто задаваемые вопросы по WireKeys

  • В: Я хочу чтобы WireKeys сохранял свои настройки в каталоге куда он установлен. Как это сделать?
    О: (1) Создайте подкаталог 'Profile' в инсталляционном каталоге. Он автоматически будет использоваться для сохранения данных. (2) Используйте ключ '-ul' в строке запуска приложения для указания каталога хранения данных (Пример: 'WireKeys.exe -ul=c:/data').

  • В: Я хочу создать ярлыки к некоторым действиям WK. Как это сделать?
    О: Используйте параметр строки "-execute" для запуска макросов или действий из bat/cmd/lnk файлов. Командная строка вида "WireKeys.exe -execute=XXX" где XXX - имя макроса/пункта быстрого запуска - запустит его на выполнение. Для выполнения действия задайте его номер предваренный символом '!', например "WireKeys.exe -execute=!15" выдвинет диск CD-Rom. Номера действий можно посмотреть в документации. Внимание: WireKeys должен быть уже запущен чтобы выполнять такие действия.

  • В: Я хочу скопировать путь к файлу, выделенному в эксплорере, в буфер обмена. Возможно ли это?
    О: Да! Для этого достаточно нажать гор. клавишу копирования в любой из дополнительных буферов. Или скопировать в обычный буфер обмена и вызвать его историю - путь к файлу будет первым в списке.

  • В: Я спрятал приложение в трей-панель , но главное меню приложения не появляется. Соответствующая опция включена. В чем дело?
    О: Некоторые приложения не имеют главного меню. Хотя панель инструментов напоминает главное меню, это не то же самое, поэтому такое меню нельзя вызвать из меню трей-панели.

  • В: Я спрятал приложение в трей-панель, но главное меню приложения отображается не полностью. Что не так?
    О: Проверьте опцию 'Show owner draw menus too' в окне настроек. Многие приложения используют такие меню, которые могут содержать графические или специальные элементы. Некоторые из таких особенностей не поддерживаются WireKeys, но большинство работают хорошо.

  • В: Я не могу убить приложение '...' из быстрого меню - заголовок приложения недоступен. Что это означает?
    О: Это означает, что вы не имеете достаточно прав, чтобы убить приложение или сервис.

  • В: Когда я пытаюсь преобразовать кодировку выделенного текста, символы получаются смешанными.
    О: Некоторые программы не поддерживают Уникод. В этом случае вы не можете преобразовать кодировку символов, или можете сделать это только частично.

  • В: Я не могу сделать снимок экрана с Универсальным Проигрывателем (Media Player) проигрывающим любое видео. На снимке я вижу только черный экран.
    О: Некоторые программы (такие как Универсальный Проигрыватель) используют специальные аппаратные средства, чтобы отображать видео с максимальной производительностью, поэтому не могут быть "сняты" без специальных действий. Чтобы включить поддержку таких окон зайдите в настройки скриншота и нажмите соотв. кнопку до того, как вы запустите желаемый видео-файл на проигрывание

  • В: Я хочу использовать WireKeys для открытия различных папок, к которым я часто обращаюсь. Как я могу это сделать?
    О: Один из путей таков: Откройте Проводник (Explorer), перейдите к каталогу, который вы хотите добавить к списку "быстрого запуска". Выберите желаемую папку/файл и нажмите Ctrl-C ( Скопировать в буфер ). После этого откройте настройки WK, добавьте новый элемент "быстрого запуска" типа 'Каталог' и кликните двойным щелчком в поле пути к приложению с правой стороны опций, как будто чтобы напечатать путь вручную. Теперь нажмите Ctrl-V (Вставка из буфера), и всё - WK автоматически преобразует ссылку в путь к папке. Можно конечно также открыть окно выбора каталога и выбрать нужную папку из дерева, но быстрей ее просто запомнить в Эксплорере. Теперь вы можете назначить быструю клавишу этому элементу быстрого запуска и испытать его на деле :)

  • В: Мне пришло письмо с ссылкой на документ, Я хочу открыть его не запуская Explorer. Возможно ли это? О: Пойдите в окно настроек и назначьте быструю клавишу для запуска выделенного текста. После этого выделите ссылку и нажмите эту быструю клавишу.

  • В: Как я могу выбрать программу для просмотра/редактирования снимка экрана по-умолчанию, чтобы двойной щелчок на jpg или bmp файле запускал выбранное мною приложение (напр. IrfanView)?
    О: WK просто выполняет команду редактировать (edit) для файла снимка экрана - если редактор не задан отдельно. Поэтому, если вы хотите выбрать определенное приложение для редактирования снимков экрана, вы можете сделать это открыв окно Проводника (Explorer), выбрав Сервис -> Свойства папки -> Типы файлов -> BMP или JPG и выбрать приложение, связанное с пунктом Edit . Так же вы можете установить редактор явно в специальном поле в окне настроек.

  • В: Мне кажется ваши дополнения (plug-ins) к программе просто не работают. В чем дело?
    О: Стандартные дополнения к программе используют системные обработчики прерываний, чтобы предоставить свои специальные возможности. Иногда они не работают без перезагрузки Windows или входа (login) в систему заново. Если вы работаете в Windows NT/2000/XP, вы можете не иметь достаточно системных привилегий. В этом случае дополнения бесполезны для вас :(

  • В: Как заставить мое приложение XXX минимизироваться в трей-панель вместо панели задач?
    О: Плагин к программе "Помошник по оконам" поможет вам. Откройте окно настроек этого дополнения и введите часть заголовка вашего приложения в поле редактирования. После этого ваше приложение будет автоматически минимизироваться в трей-панель каждый раз когда вы покидаете его. Например для того, чтобы минимизировать окна 'Internet Explorer' в трей, вы можете напечатать '*Explorer*'

  • В: Я хочу чтобы WireKeys сохранял свои настройки внутри каталога с программой, а не в "Documents and Settings/.../Application Data/WireKeys". Это возможно?
    О: Да. Используйте ключ командной строки "-ul" чтобы заставить WireKeys использовать нужный Вам каталог для сохранения своих данных
    Примеры:
    1) WireKeys.exe -ul
    WireKeys будет сохранять настройки в "UserData" внутри каталога с WireKeys.exe
    2) WireKeys.exe -ul=XXX
    WireKeys будет сохранять настройки в "XXX" внутри каталога с WireKeys.exe
    3) WireKeys.exe -ul="C:\XXX"
    WireKeys будет сохранять настройки в "C:\XXX"


  • В: Два вопроса о текстовых (клавиатурных) макросах. Во-первых: Как я могу поместить символ конца строки в вывод макрокоманды? Я хотел бы выводить многострочный текст. Во-вторых: Как я могу перенаправить вывод макрокоманды в определенные окна. Я могу иметь 4 открытых окна, но хочу, чтобы моя макрокоманда знала в которое окно выводить текст
    О: Это - пример того, как создать многострочный ответ макрокоманды и перенаправить его в указанное окно (которое должно быть уже открытым): 
    // Шаг 1 из 3
var OUTPUT='';
// Шаг 2 из 3
OUTPUT+='Первая строка многострочного текста ';
OUTPUT+='\r\n';//Добавление символа конца строки
OUTPUT+='Вторая строка многострочного текста ';
OUTPUT+='\r\n';//Снова добавление символа конца строки
OUTPUT+='Последняя строка многострочного текста ';
// Активизация окна
// В этом примере будет активировано окно Блокнота (Notepad) 
// с безымянным документом
var WshShell = new ActiveXObject('WScript.Shell');
WshShell.AppActivate( Untitled - Notepad );
// Шаг 3 из 3
OUTPUT;
    И помните, что макрокоманда может выполнить различные действия в зависимости от её параметров. Вы можете использовать один файл макрокоманды в 4 различных макросах и устанавливать их параметры и быстрые клавиши независимо. Например, можете сделать заголовок окна параметром для каждой макрокоманды. В этом случае строка 'WshShell.AppActivate( Untitled - Notepad );' должна быть изменена на 'WshShell.AppActivate(unescape( %PARAMETER ));'

  • В: Я включил опцию 'Integrate with WireNote', и иконка в трей-панели внезапно исчезла. Как я могу её вернуть?
    О: Вы можете вызвать настройки WireNote из подменю 'Utilities' меню иконки WireNote в трей-панели.

  • В: Я хочу видеть вашу программу на моем языке. Это возможно?
