WireNote FAQ

  • Q: I want WireNote to save its data in the program`s installation directory. What should i do?
    A: (1) Create folder named 'Profile' in the installation directory. It will be automatically used to save all persistent data. (2) You can add '-ul' command line switch to application shortcut to define directory for saving application`s data (Example: 'WireNote.exe -ul=c:/data').

  • Q: I hide my note accidentally and can`t find it again.
    A: You can find it at any time in main window or in tray menu under corresponding submenu

  • Q: I want to use this program instead of Messenger service/WinPopup. Is it possible?
    A: Yes. WireNote will ask you if you want to stop Messenger Service or, if you have WinPopup-like applications, you must stop them manually before running WireNote. After that you can send and receive such messages through WireNote.

  • Q: I can`t receive messages from other users who use Winpopup/Messenger service. What`s wrong?
    A: If you have more than one net-card or other special net environment, default networking values used by WireNote are not sufficient. You can try to change LANA option for Netbios protocol. These setting can be tuned in Preferences dialog

  • Q: Why can't i send/receive notes to/from other computers through XPC protocol?
    A: Sending/receiving messages through XPC protocol to/from other computers works only in a local TCP/IP network with static IP addresses. In addition sender and recipient both must use the same port number and the recipient must accept incoming notes. These settings can be defined in Preferences dialog.

  • Q: If I have small home LAN and I want my reminders/Notes/ToDos, created on one machine, to be active on all other computers, can i do such a thing?
    A: Yes, you can. To do this, create shared folder (which will be accessible from all computers) and type it in the preferences dialog under "Billboard" section. After that you can publish items like reminders/notes and ToDos at billboard and all computers conected to this folder will receive them and all changes to published items


  • Q: How can i backup my notes, ToDos, reminders and settings?
    A: Open preferences dialog and click "Open folder with all saved settings ...". You can zip this folder for future use. To restore just unpack zip over the same location


  • Q: I heard that this program can save me from junk emails aka SPAM in my mailboxes. Is it true?
    A: Yes. You must set your mailbox account properties and after that WireNote will clean up your mailbox occasionally. You have to define which message strings mark it as spam, but you are responsible to set this strings properly - this mean that such strings must NOT be met in the normal emails. It is highly recommended to set WireNote in training mode, so you will see all received emails and will be able to train WireNote to recognize SPAM. WireNote saves a lot of internet traffic, cos messages are NOT downloaded entirely. Only first lines of every message will be transferred to your computer from your mailboxes. WireNote supports POP3 mailboxes only

  • Q: I am using WireNote for messaging and my collegues receive message with strange strings like "$=".
    A: This is because your collegues are using previous versiong of WireNote. Please, advise them to upgrade :) Or you can disable features not supported by previous versions of WireNote in the preferences ("Protocols" options)
