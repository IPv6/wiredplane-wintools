Note/Message Interface

Note is the small colored window with simple edit field that stays on your desktop. You can store any piece of the text information in it. All notes (but not messages by default) are saved between computer shutdowns. At startup every note will be loaded only if it is marked as "Keep forever" or is not older than specified in preferences.

One of the useful teatures is that you can attach note to desktop or to any application window. In the last case note will be hidden when your will work with other applications. You can attach note to several windows simultaneously by defining title patterns in advanced options dialog. Possible patterns are:
[?...text...?] - Any window containing "text" in the title
[!...text...!] - Any window not containing "text" in the title
[=...text...=] - Any window with the title "text"

Note can be switched between 4 views:
- Minimal. In this view you will not see any buttons and extra fields - just note title and text
- Buttons. The same as minimal but with some common used functions binded to special buttons
- Extended. The most functional view, used for messages. In comparison with Buttons view there are additional buttons and fields for messaging
- Iconic. The smallest view, only note/message icon and title will be visible

You can drag`n`drop files from other applications to attach dragged files to this note. If you want to open dragged file as text (instead of attach action), hold Shift key while dropping such file
