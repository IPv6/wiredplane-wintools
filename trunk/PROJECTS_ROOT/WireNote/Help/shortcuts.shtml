Keyboard Shortcuts

Note/Message 
Ctrl-Shift-NNew note (hotkey can be changed through preferences dialog)
Ctrl-Shift-MNew message (hotkey can be changed through preferences dialog)
Ctrl-SSave note
EscapeHide note (corresponding option must be ON)
Ctrl-HHide note
Ctrl-DDelete note (to trash, if corresponding option is ON)
Ctrl-ASelect all and copy to clipboard
Ctrl-MSwitch note to message mode
Ctrl-ISwitch note between icon and full mode
Ctrl-ESwitch read-only flag (prevent from edit)
Ctrl-NSwitch auto hide mode
Ctrl-WSwitch word wrap mode
Ctrl-TSwitch topmost mode
Ctrl-USnap size to text
Ctrl-KEnable/disable clipboard spy
Ctrl-OShow advanced options dialog
Ctrl-BPublish in the common section of billboard
Ctrl-EnterSend message
Ctrl-Enter-ShiftSend message and delete if delivered successfully
Ctrl-QQuote note/message text
Ctrl-LShow current user`s message history
Ctrl-Shift-LLoad text from file
Ctrl-Shift-SSave text to file
Ctrl-Up/Ctrl-DownList of the recent recipients
Ctrl-1,2,3...0Reply message with quick answer number 1,2,3..0
Ctrl-Shift-1,2,3...0Remember selected as quick answer number 1,2,3..0
ToDo 
Ctrl-Shift-TNew ToDo (hotkey can be changed through preferences dialog)
Ctrl-EnterSave and close
Ctrl-SSave ToDo
Ctrl-OShow advanced options dialog
Ctrl-DDelete ToDo (to trash, if corresponding option is ON)
Ctrl-BPublish in the common section of billboard
Reminder 
Ctrl-Shift-RNew Reminder (hotkey can be changed through preferences dialog)
Ctrl-EnterSave and close
Ctrl-DDelete Reminder (to trash, if corresponding option is ON)
Ctrl-BPublish in the common section of billboard
Main window 
Ctrl-Alt-TShow main window (hotkey can be changed through preferences dialog)
Ctrl-SSave all
Ctrl-TSwitch topmost mode
Ctrl-OShow preferences dialog
EscapeHide to tray
InsCreate new item (depending on selected folder)
EnterModify/Activate selected item
SpaceChange selected item state
DelDelete selected item
Address book 
Ctrl-Alt-AShow address book (hotkey can be changed through preferences dialog)
Ctrl-LShow message history
Ctrl-OShow preferences dialog
EscapeHide to tray
InsCreate new person
EnterModify selected person
DelDelete selected person
