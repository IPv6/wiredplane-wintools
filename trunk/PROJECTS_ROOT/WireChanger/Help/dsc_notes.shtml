Desktop Notes

This handy feature helps you to organize your schedule and to keep your ideas in a safe yet easily-accessible place - your desktop. To add a note, just double-click the desktop wallpaper and type your text in the newly created window. To edit the note, just double-click it again. All notes can be easily accessed through tray menu too. You can also choose a note icon and set its background color and text color. Another insteresting feature - reminding. You can set up reminder (it can be also reccurent) for note - just click icon in the note title. All reminders can be viewed (and modified) on calendar/minilendar widget as well or in separate Calendar (tray menu - calendar).
Some keyboard shortcuts to quickly set some note parameters:

Ctrl-Enter/Ctrl-S: Add note to desktop
Ctrl-T/Ctrl-B: Note text/background color
Ctrl-Space: Change view (buttons/text only)
Ctrl-Up/Ctrl-Down/Ctrl-I: Change note icon
Ctrl-R: Add/Modify reminder
Ctrl-D: Delete note
- To quickly change note icon you can click system icon of the note window (in the top-left corner).
- In the "text only" view you can use additional buttons in the title area to save and delete note
