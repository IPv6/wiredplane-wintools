Mouse gestures plugin

Recognition

For recognition of mouse gestures you must press right mouse button with Alt key during moving a mouse (You can change this combination in plugin options). For example for recognition "left" gesture, press Alt, right mouse button and move a mouse to the left. If a neural network can recognize the gesture, then you will see the name of recognized gesture.

Attention: the direction is very important.
For example, you can draw the "circle" gesture via thousand different ways, but valid way is: press mouse button and move a mouse to the right and down and so on. Once more: it's gesture but not 2D image. If you draw gestures in different directions, they may not be recognized
