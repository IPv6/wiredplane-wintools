Preferences Dialog

To access the WireChanger Preferences screen, right-click the WireChanger tray icon and select Preferences from the popup menu. By default, the Common preferences are shown first. You can also access this screen by clicking the Preferences button on the Wallpapers list window. When you access Preferences this way, the Wallpapers list options are shown first.

Common
  • Start with Windows (Checkbox) - When checked, WireChanger will run at Windows startup.
  • User interface language (Dropdown) - Selection detemines the language displayed within WireChanger. Choose from English, Russian, Spanish, Italian, Arabian, and more as they become available.
  • When program starts (Dropdown) - Selection determines what will happen to the current wallpaper when WireChanger starts. Choose from 'Do Nothing', 'Wait and Change', or 'Wait, Change and Exit'. 'Change (keep period between restarts)' - If you close WireChanger (by exit or shutdown) 15 mins prior to next change, WireChanger will save this timeout for next run (and will wait 15 min after start before changing wallpaper).
  • Tray icon appearance (Dropdown) - Selection determines whether or not a WireChanger icon will appear in the Windows taskbar tray and what that icon can look like. Choose from 'Simple icon', 'Minimized wallpaper', or 'No icon at all'.
  • Action to perform on 'Close' button (Dropdown) - Selection determines what will happen when you click the Close (X) button at the top right corner of the Wallpapers list window. (This action does not occur when you click the Close button of the Properties window.) Choose from 'Ask user', 'Close program', or 'Minimize to tray'.
  • Blink tray icon while processing wallpaper (Checkbox) - When checked, the WireChanger tray icon will flash briefly as the wallpaper changes.
  • How to fill borders around images (Dropdown) - When an image has an aspect ratio different from that of the screen resolution, this selection determines how the remaining gaps should be filled in. Choose from 'Fill with background color', 'Use image edges infinitely', or 'Use mirrored image infinitely'.
  • Blend borders around wallpaper with background color (Checkbox) - When checked, wallpapers that do not fill the entire screen will blend seamlessly with the background color.
  • Wallpaper background color (Ellipsis button) - Used in connection with the previous option, when 'Fill with background color' is chosen, to determine what that background color will be. Brings up the standard color selection window.
If you do not see the following Hotkeys, Exceptions, Warnings, and Miscellaneous options, click the blue triangle in the tree structure next to the Common option. Clicking this toggle will alternately expand and contract the options below Common. The same is true for the triangle toggle by the Wallpapers list.

Hotkeys

To change any hotkey setting, double-click the Value area for the given action. This enables a dropdown list from which you can optionally choose to add a (special) key, add a modifier, or clear the current hotkey value. Click the X button next to the dropdown list to clear the hotkey value.

Special keys include Space, Page Up, Page Down, End, Home, the four arrow keys, Sys Req, Insert, Delete, Apps, any key in the keypad group (except Enter), any function key (F1-F12), Num Lock, and Scroll Lock.

Modifiers include Shift, Ctrl, Alt, and Windows.

Special keys and modifiers are not required to setup a hotkey. With the dropdown list enabled, you can simply assign a single letter from the main section of the keyboard ('C' or 'N', for example) as the hotkey.

Available hotkey actions and their default settings:
  • Change wallpaper - Alt-W
  • New desktop note - not defined
  • Show list of images - not defined
  • Next wallpaper - not defined
  • Prev(ious) wallpaper - not defined
Privacy/Exceptions
  • When screensaver is running or computer is locked (Dropdown) - Selection determines how WireChanger should handle its normal functions under these more unusual conditions involving screensavers and purposely locked systems. Choose from 'Do not change wallpaper', 'Change in normal way', or 'Change only once'.
  • Don't change if memory usage threshold exceeded (percent) (Checkbox) - When checked, WireChanger will not change wallpapers if your system's memory usage has exceeded the value, expressed as a percent, that you have entered. The default percentage is 80.
  • Don't change if CPU usage threshold exceeded (percent) (Checkbox) - When checked, WireChanger will not change wallpapers if your system's CPU usage has exceeded the value, expressed as a percent, that you have entered. The default percentage is 60.
To change the percentages described above, double-click each value box. You can enter whole numbers ranging from 0 to 100 percent.

System options
  • Load directories with subfolders (Checkbox) - When checked, if you add an entire folder of images to the WireChanger rotation and that folder contains sub-folders, images in those sub-folders will also be included in the rotation.
  • Set last wallpaper for default user (Win NT/2k/XP) (Checkbox) - When checked, if a default (guest) user logs on to the system, the wallpaper that was last shown undeer your login will be the wallpaper displayed for the guest.
  • Track wallpaper changes and apply notes automatically, Apply active templates also (Checkbox) - When checked, WireChanger is aware of your attempts to change the wallpaper from another program (that has a 'set as wallpaper' option) and automatically updates the new image with any notes and templates you may have had on the existing wallpaper.
  • Generate wallpapers in low priority (Checkbox) - Changing to a new wallpaper can sometimes use much of the CPU's resources for a short period of time. When checked, such CPU usage will be smoothed out over a longer period of time making the shortage of resources less noticeable but making the image change take a little longer.
  • How to fill desktop with patterns (Dropdown) - If the current wallpaper is smaller than your screen resolution, this selection determines how Wirechanger will handle the 'extra' space on the desktop. Choose from 'Center/Stretch', 'Tile', or 'Mirror tiling'.
  • Set icons transparency and text color (Checkbox) - When checked, WireChanger will attempt to remove the background from desktop icon labels (i.e. make it transparent) and change the color of the text to the one you specify here. Click the ellipsis button to bring up the standard color selection window. Under certain conditions, this may not work due to a bug in Windows.
  • Distance to stick windows together (pix) - Double-click the value box to change the pixel number. If two WireChanger windows are placed on the screen this distance apart (or less), they will automatically lock edges.
  • Check for updates every ... day(s) (Checkbox) - When checked, WireChanger will check for newer versions at WiredPlane and popup a notice when one is available. Set the frequency in days to check the website by double-clicking the value box.
  • Wallpaper file - This option has two possible uses when checked. 1) If your browser's home page is a file on your hard drive (as opposed to a page on the Web), the image in this option will be the background of that page. 2) When connected to a LAN, you can enter the path to an image in a shared folder. Then any computer with access to that shared folder can use that same image as their wallpaper.
  • Wallpaper exchange service (Dropdown) - Determines whether you want to include online wallpapers in the rotation. Choose from 'Suggest exchange sometimes', 'Approve wallpaper exchange', or 'Turn this service off'.
Confirmations

  • Skip information warnings (Checkbox) - When checked, suppresses notices (such as the number of new images added when you choose to add an entire folder of them) while working in the main Wallpapers list window.
  • Skip warning about broken image link (Checkbox) - When checked, suppresses notices about missing images. If you added images to your WireChanger rotation and later deleted some of them from your hard drive, WireChanger has no way of knowing that you removed them. (Note that each theme has an option to 'remove broken images' to help prevent this situation.)
  • Skip warning about Active Desktop enable/disable (Checkbox) - When checked, suppresses notices about the need to enable and disable the Active Desktop when working with the Clocks Template.
  • Skip warning about refreshing image list (Checkbox) - When a large number (3000 or more) of images is added to the WireChanger rotation, it can take a long time to process them. When checked, this option suppresses the notice that processing will take longer than usual and that the user should be patient.
  • Skip warning about applying changes (Checkbox) - When checked, suppresses notices generated when you click the Apply button, when you save a sticky note, and when you change settings in the active template.
  • Skip warning about deleting theme (Checkbox) - When checked, suppresses notices generated when are about to delete a theme.
  • Skip warning about deleting clock (Checkbox) - When checked, suppresses notices generated when are about to delete a clock.
  • Show tray popup in difficult situation (Checkbox) - When checked, if WireChanger attempts to change the wallpaper during a time of high CPU usage or low memory resources, not only will the change fail but the popup menu will appear by the tray icon as well.
  • Warning window: Place button on taskbar instead of tray icon (Checkbox) - When checked, WireChanger will be minimized to the taskbar, and no icon will show in the tray. Warning windows then would naturally pop up near the taskbar button.
