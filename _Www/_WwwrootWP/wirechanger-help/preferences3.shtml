Wallpaper Templates

Templates are additions to your desktop that look and feel like part of the wallpaper itself. Enable/disable a template by checking/unchecking the associated checkbox. Click the 'Change settings of...' link to open a new window displaying options for that template. Click the 'Make a copy of...' link to have another set of checkbox-and-links automatically insert itself below the current set. The last link in the new set will be 'Remove...' which you can use to delete the new set. You can have more than two copies of a theme. Each new set of checkbox-and-links is appended with a number (starting with [2]) to avoid confusion over which is the original and which are the copies. You can select any or all of the templates below. To make changes take effect, you may have to manually rotate (Alt-W) to the next wallpaper.

Latest CNN News

This template consists of a news area and a heading area that always says 'CNN News'. All options affect both areas except 'Show border and background rectangle' which only affects the news area. To remove the border and background from the heading area, set the 'Background opacity' option to 0. Note that you must be online to use this template. If you have a firewall in place, you may be alerted that WireChanger is attempting to access the Internet.
  • News font - Brings up the standard font selection box for choosing font type and size.
  • Text color - Brings up the standard color selection box for setting the text color.
  • Background color - Brings up the standard color selection box for setting the background color.
  • Background opacity (percents) - Set the opacity of the news area ranging from 0% (transparent) to 100% (opaque). The default is 40%.
  • Show border and background rectangle (Checkbox) - When checked, a border and background will appear with the text. Affects only the news areas.
  • Show shadow around news (Checkbox) - When checked, individual text characters will be outlined which can make them more visible relative to the background or wallpaper.
  • Open news page on double-clicking news - When checked, a browser page will open containing the full news story.
  • CNN News size and position - Double-click the value box to manually set the...
    • X Position (pixels) - ...location of the left border of the news template.
    • Y Position (pixels) - ...location of the top border of the news template.
    • Width (pixels) - ...width of the news template.
    • Height (pixels) - ...height of the news template.
    • Set position - Click to make the news template appear as a window that you can drag and resize. This option is usually easier to use than the previous four manual settings.

    • Edit annual events - Click to open a window in which you can add, change, or delete events that appear in the subtitle.
  • Calendar size, color and position
    • Text color - Brings up the standard color selection box for setting the text color in calendar cells.
    • Cell background color - Brings up the standard color selection box for setting the color of 'normal' cell backgrounds.
    • Weekend color - Brings up the standard color selection box for setting the color of Saturday and Sunday cell backgrounds.
    • Holidays color - Brings up the standard color selection box for setting the color of holiday cell backgrounds.
    • Calendar opacity (percents) - Set the opacity of the calendar ranging from 0% (transparent) to 100% (opaque). The default is 40%.
    • Text font - Brings up the standard font selection box for choosing calendar cell font type and size.

Image mosaic

This option creates a tiled pattern using the current wallpaper image.

  • Rows to show - Double-click the value box to set the number of rows in the mosaic. The minimum is 1; the maximum is 100. The default is 2.
  • Columns to show - Double-click the value box to set the number of columns in the mosaic. The minimum is 1; the maximum is 100. The default is 2.
  • Offset cells on X axis (percents) - Double-click the value box to set the amount, measured as a percentage from 0 to 100, that the rows of cells should shift. The default is 0.
  • Offset cells on Y axis (percents) - Double-click the value box to set the amount, measured as a percentage from 0 to 100, that the columns of cells should shift. The default is 0.
  • Mirror images in the cells (Dropdown) - Choose from 'Do not mirror images', 'Mirror from second cell' (default), 'Mirror first cell by X axis', 'Mirror first cell by Y axis', Mirror first cell by X and Y axes', or 'Random'.
  • Use random images for every cell except first (Checkbox) - When checked, only the cell in the top left corner of the mosaic is guaranteed to contain the current image in the standard rotation. Other cells can contain any image in the rotation.
  • Effects frequency (percents) - Double-click the value box to set the amount, measured as a percentage from 0 to 100. The default is 100, which will generate the selected effects every time the background changes. A setting of 50 would affect every other page.
Memo pad

  • Edit birthdays - Click to open a window in which you can add, change, or delete birthdays.
  • Show interesting events for this day (Checkbox) - When checked, if an interesting event falls on the current date, it will be displayed on the memo pad. You get to determine what an 'interesting' event is.
  • Edit interesting events - Click to open a window in which you can add, change, or delete events.
  • Background color - Brings up the standard color selection box for setting the color of the memo background.
  • Foreground color - Brings up the standard color selection box for setting the color of the memo text.
  • Memo pad opacity (percents) - Set the opacity of the memo pad ranging from 0% (transparent) to 100% (opaque). The default is 50%.
  • Edit memo text on double-clicking memo pad (Checkbox) - When checked, you can update the your notes on the memo pad by double-clicking it.
  • Memo pad font - Brings up the standard font selection box for choosing memo pad font type and size. Default is Arial, 10pt.
  • Edit memo note - Opens Notepad where you can add, change, or delete notes for the day. This is the alternate method to use if the edit on double-click checkbox (above) is not checked.
  • Memo pad size and position
    • X Position (pixels) - ...location of the left border of the memo pad.
    • Y Position (pixels) - ...location of the top border of the memo pad.
    • Width (pixels) - ...width of the memo pad.
    • Height (pixels) - ...height of the memo pad.
    • Set memo pad position - Click to make the memo pad appear as a window that you can drag and resize. This option is usually easier to use than the previous four manual settings.

Image mixer

The Image Mixer combines two desktop pictures to create unique effects. The options available are difficult to describe, so we suggest you simply play around with them to see what they do and to generate an effect that you like.

Second picture

The Second Picture template adds another image (not mixed with the original) to your desktop. A few common options are available for setting the opacity, border, size, and position of the picture.

Special Effects

Special Effects can be used to alter the appearance of the main desktop image. The options available are difficult to describe, so we suggest you simply play around with them to see what they do and to generate an effect that you like.

Web camera

The Web Camera template adds shots from the Web camera of your choice to the desktop.

  • Link to Web camera image - Double-click the value box to enter the URL of the Web camera. The default is random thumbnail images at WiredPlane.
  • Web shot opacity (percents) - Set the opacity of the Web camera image ranging from 0% (transparent) to 100% (opaque). The default is 100%.
  • Prevent image from being cached (Checkbox) - When checked, Web shots will not be cached on your machine. This ensures that only new images from the Web are being displayed.
  • Open corresponding Website on double-clicking icon (Checkbox) - When checked, you can double-click the Web camera image to go to the Website where that image is located.
  • Show border around Web shot (Checkbox) - When checked, a border will appear around the Web camera image.
  • Web shot border color - Brings up the standard color selection box for setting the color of the Web camera image border.
