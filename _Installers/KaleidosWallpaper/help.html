KaleidosWallpaper

This is simple wallpaper generator, based on interesting effect called Kaleidoscope. The only thing you should set up before first run is image directory. By default it is "My pictures" folder, but you may want to set it to your local gallery path. KaleidosWallpaper can work on almost any images, so don`t worry.
This software is absolutely free for everyone. But if you find it interesting you can donate us for future development (Donate!).
KaleidosWallpaper does not contain any spyware or adware of any kind.
If you want more effects or something useful on top of your wallpaper (like interactive calendar with clickable cells or clock), we recommend you to try our WireChanger. This is full-featured wallpaper manager with a lot of preinstalled useful widgets and sophisticated randomizing options.
